# gmn
Dead Simple Terminal Gemini Client to stop opening the browser

# Features
- Gemini API.
- Syntax Highlithing.
- Web-like result output.
- No config required.
- Multiline input.

# Usage
This works like an AI Shell like Warp but without the extras.

```bash
gmn
```

Type your prompt. and press `Ctrl + d` when you finish.
